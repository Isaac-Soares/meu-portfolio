## `> ARCHITECTURE_NOTE`

> One decision I'm particularly proud of in IsaacOS: instead of fighting with iOS WebKit's scroll limitations inside nested `position: fixed` elements, I designed a **data-driven mobile redirect system**. The `js/data.js` file acts as a single source of truth тАФ both the desktop OS and the dedicated `mobile.html` consume the same data. Add a certification once, it appears everywhere.
>
> This is the kind of architectural thinking I bring to every project.
