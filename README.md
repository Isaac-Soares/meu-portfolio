## `> PROJECTS`

### 🖥️ IsaacOS — Interactive Developer Portfolio
> *The project you're looking at right now.*

A retro-style Operating System portfolio built entirely with vanilla web technologies. No frameworks. No shortcuts.

```
Architecture highlights:
  ├── Canvas-based 2D room animation (requestAnimationFrame loop)
  ├── Data-driven design — js/data.js feeds both desktop & mobile
  ├── Responsive detection → auto-redirects to dedicated mobile.html
  ├── Pixel-perfect retro UI with custom window management system
  └── Cross-origin click detection mapped to original image coordinates
```

**Stack:** `HTML5` `CSS3` `JavaScript` `Canvas API`

## `> ARCHITECTURE_NOTE`

> One decision I'm particularly proud of in IsaacOS: instead of fighting with iOS WebKit's scroll limitations inside nested `position: fixed` elements, I designed a **data-driven mobile redirect system**. The `js/data.js` file acts as a single source of truth — both the desktop OS and the dedicated `mobile.html` consume the same data. Add a certification once, it appears everywhere.
>
> This is the kind of architectural thinking I bring to every project.
